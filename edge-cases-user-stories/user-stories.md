# 👤 User Stories

- As a student, I want to know what careers fit me so I can choose a direction
- As a beginner, I want simple steps so I can start without feeling overwhelmed
- As an experienced student, I want advanced guidance to level up
- As a non-technical user, I want relevant options that match my strengths
- As a confused student, I want clarity and structure
- As a motivated student, I want actionable next steps immediately
